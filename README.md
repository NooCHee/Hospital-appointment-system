# Hospital-appointment-system
#1st grade of comp eng

This is a hospital appointment booking system used for various types of tests.
in this java console based program we can book laboratory tests, it will give us a receipt in the end that includes patient's name, age, phone number, gender, blood type, and type of chosen test, and with a discount for people that are 60 years and older, or kids that are 10 years or younger.
the program continues running until user chooses exit option.
